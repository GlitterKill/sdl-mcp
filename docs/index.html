SDL-MCP

Cards-First Context for Coding Agents

SDL-MCP indexes repositories into symbol cards and graph edges so agents can retrieve focused context before requesting raw code.

What SDL-MCP Does

SDL-MCP (Symbol Delta Ledger MCP Server) provides token-efficient code context via a ladder: symbol search, symbol cards, graph slices, skeletons, hot paths, then policy-gated code windows.

Lower Token Cost

Cards and slices avoid loading full files by default.

Better Relevance

Dependency-aware graph traversal returns related symbols.

Safer Access

Policy controls gate large code windows and capture intent.

Quick Start

# Install
npm install -g sdl-mcp

# Initialize config in your repo
sdl-mcp init --client codex

# Validate environment and index
sdl-mcp doctor
sdl-mcp index

# Run MCP server
sdl-mcp serve --stdio
