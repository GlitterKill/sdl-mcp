The upstream @ladybugdb/core npm package ships 496MB (all platform binaries + full C++ source + 322MB test datasets). We created split packages under @sdl-mcp/ladybug-{platform}-{arch} so npm only downloads the ~13-25MB binary for the user's platform. The wrapper @sdl-mcp/ladybug resolves the correct platform binary at runtime via require.resolve(). CI workflows updated to handle both upstream kuzu and split packages.
