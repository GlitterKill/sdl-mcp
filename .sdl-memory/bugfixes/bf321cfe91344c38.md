CI failures on 2026-03-26 came from two coupled issues in scripts/real-world-benchmark.ts. collectCandidateFiles had an unclosed array spread around globSync, which Node 24 reported as ERR_INVALID_TYPESCRIPT_SYNTAX when benchmark:matrix executed source TS directly, and fixing that exposed a second bug where the helper referenced ignore instead of the exclude parameter. A focused smoke test that spawns scripts/real-world-benchmark-matrix.ts with --limit-runs 1 --skip-index now guards both the parse path and the helper call site.
