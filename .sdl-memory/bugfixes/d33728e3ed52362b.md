Memory files are written to .sdl-memory/<type>/<memoryId>.md. When memoryId comes from caller-supplied input (update path) or YAML frontmatter parsing, a crafted value with path separators (../../) could escape the directory. Added MEMORY_ID_PATTERN regex (/^[0-9a-f]{8,64}$/) and validateMemoryId() function. Validation added in writeMemoryFile (before path construction) and parseMemoryFileContent (after frontmatter extraction).
