A 2026-03-25 review fix set addressed three drift patterns. Binary-file skips in processFile must call persistSkippedFile before returning so stale symbols and edges are removed when a text file becomes binary. waitForIdle-style APIs in live-index code must not unref the timer that resolves their completion/timeout promises, or an otherwise idle process can exit before the awaited flush finishes. Config defaults were standardized on the typed config/policy engine contract: allowBreakGlass=false and prefetch.enabled=false across serve defaults, MCP schemas, init scaffolding, the sample config, and regression tests.
